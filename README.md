# Habit Tracker

A modern, responsive weekly habit tracker built with React, Vite, Tailwind CSS and LocalStorage persistence.

Designed as a frontend engineering assessment project focused on usability, accessibility, architecture, and thoughtful interaction design.

---

## Features

* Add, rename, and delete habits
* Weekly habit tracking grid
* Previous and next week navigation
* Back to current week shortcut
* Persistent historical tracking
* Current streak calculation
* Responsive mobile-first layout
* LocalStorage persistence

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS

## Design Philosophy

The interface prioritizes:

* Fast weekly scanning
* Clear visual hierarchy
* Low cognitive load
* Satisfying interaction feedback
* Strong responsive behavior

The weekly grid was intentionally designed to resemble lightweight productivity SaaS dashboards rather than generic CRUD tables
